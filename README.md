### What is this?
TernLink helps Kenyan youth find real online gigs without getting scammed. 

If you’ve ever seen those "Earn 20K daily typing!! Pay 500 registration" WhatsApp messages... yeah, this project is the opposite of that. Only verified gigs, plus a whole page teaching you red flags.

### Why I built it
In order to connect the youth to viable jobs they can do as they proceed with their University studies

### How it works
1. *Home page:* Pick your skill + type your location
2. *Find Gigs page:* Shows gigs that match what you picked. Uses JS to create them
3. *Safety Center:* Insight common scams in Kenya + what to do if you got scammed
